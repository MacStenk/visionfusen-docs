## Was ist VisionFusen?

VisionFusen signiert deine Bilder kryptografisch auf dem **Nostr-Protokoll**. Das Ergebnis ist ein unveränderlicher, dezentraler Nachweis der Urheberschaft.

## Features

- **NIP-94 Signierung** - Standardkonform nach Nostr Improvement Proposal 94
- **CDN Upload** - Bilder werden auf cdn.visionfusen.org gehostet
- **Multi-Relay** - Events auf 4 Relays (damus, primal, nos.lol, visionfusen)
- **LLM-optimiert** - Statische HTML-Seiten mit Schema.org für AI-Sichtbarkeit
